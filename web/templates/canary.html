Canary Tokens

Settings

When a client registers, Duckway auto-generates canary tokens via canarytokens.org. If an attacker steals these decoy credentials and uses them, you'll get an email alert.

How It Works

TypeWhat gets deployedWhere on client
aws_keys Fake AWS access key + secret key ~/.aws/credentials
github Fake GitHub personal access token ~/.config/gh/canary_token

Canary tokens are placed in realistic paths but NOT in keys.env, so AI agents use the real placeholder keys. Attackers who steal files from the machine will find these decoys and trigger an alert when they use them.
